## Usage


```markdown
Blank lines are required both before and after the <Collapse> and </Collapse> tags.

HERE CONTENT

</Collapse>
```

## Example

Expand and collapse:

<Collapse hidden title="Foo">

   ``` sh
   echo "foo"
   ```

</Collapse>
